# Straightline Creative Brand Book

## Brand Promise

Creating Content Worth Watching.

## Brand Voice

Personal. Thoughtful. Curious. Human.

Straightline Creative should sound like a real conversation, not a sales page. The writing should feel clear, warm, confident, and specific.

## Writing Rules

1. Write like we are having coffee with the client.
2. If it sounds like marketing, rewrite it.
3. Every sentence needs a purpose.
4. Every section should end with a memorable thought.
5. Every call to action should feel like an invitation.
6. Respect the audience's time.
7. Avoid corporate language.
8. Avoid using dashes in visible copy. Rewrite the sentence so it flows naturally.

## Brand Hierarchy

The brand is Straightline Creative.

The voice is Kevin.

The promise is Creating Content Worth Watching.

## Core Beliefs

Tools change. Storytelling doesn't.

Every great project starts with a conversation.

Technology is a tool. Curiosity is the advantage.

The medium changes. The mission doesn't.

The best stories are still built by people.

## Emotional Goal

When someone leaves the site, they should not just think the site looks good. They should feel like Straightline Creative understands story, cares about the work, and would be a thoughtful creative partner.
